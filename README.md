# FoolUi

> 基于vue2.x，可复用UI组件

## Install

``` bash
npm install fool-ui
```
## Quick Start
``` bash
import Vue from 'vue'
import Fool from 'fool-ui'
 
Vue.use(Fool)
 
// or
import {
  Button
  // ...
} from 'fool-ui'
 
Vue.use(Button.name, Button)
```
## Browser Support
``` bash
Modern browsers and Internet Explorer 10+.
```
